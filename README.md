## Installation

```

```

## Contribute

### Install Yarn

```
brew update
brew install yarn
```

### Install packages
```
yarn install
```

## Start Dev-Server

```
yarn start
```

### Install commitizen

```
npm i -g commitizen
npm i
```

### Commit changes

```
yarn run commit
```
