Revealer

Every Animation can have the following properties

animation string The Revealer animation that should be used
singleAnimation Boolean Whether the animation should be run multiple times when scrolling through or not (default = multiple times)

BlockReveal Animation

Allowed properties:

easing Easing See easings
easeIn Easing Overwrites easing for the initial animation of the overlay block
easeOut Easing Overwrites easing for the second animation of the overlay block
duration number in ms
delay number in ms
bgColor string css color
measureContent Boolean if set, measures inner content size instead of spanning the whole width / height of the block
